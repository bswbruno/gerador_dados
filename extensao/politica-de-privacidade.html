Política de Privacidade

Extensão: Gerador de Dados de Teste

Última atualização: 23 de junho de 2026

Esta extensão não coleta, armazena, transmite nem compartilha nenhum dado pessoal do usuário. Tudo funciona localmente no seu navegador.

1. O que esta extensão faz

O Gerador de Dados de Teste é uma ferramenta voltada para desenvolvedores e profissionais de QA (qualidade de software). Ela permite gerar dados sintéticos como CPF, CNS, nomes fictícios e textos aleatórios para uso em ambientes de teste, além de oferecer recursos de contagem de caracteres e leitura de limites de campos de formulário.

2. Dados coletados

Nenhum dado pessoal é coletado. A extensão não identifica, rastreia nem monitora o usuário de nenhuma forma.

Os únicos dados armazenados são:

  • Preferências de configuração (ex.: tema de cores, formato de CPF, métricas de contagem ativas), salvas exclusivamente no armazenamento local do navegador (chrome.storage.local).
  • Histórico de dados gerados (ex.: últimos CPFs ou CNS gerados), também salvo apenas localmente e acessível somente pelo próprio usuário.

Nenhuma dessas informações sai do dispositivo do usuário.

3. Acesso a páginas web

A extensão solicita permissão de acesso a todas as URLs (<all_urls>) para poder injetar funcionalidades nas páginas abertas pelo usuário, como preencher campos de formulário com dados gerados e exibir a contagem de caracteres de textos selecionados.

Esse acesso é estritamente local e não é usado para leitura, captura ou envio de conteúdo das páginas para servidores externos.

4. Dados enviados para servidores externos

Nenhum dado é enviado a servidores externos.

A extensão opera completamente offline e não realiza chamadas de rede, com exceção do clique nos ícones de Sugestões e Avaliar, que apenas abrem links externos em nova aba quando acionados manualmente pelo usuário.

5. Compartilhamento com terceiros

Nenhum dado é compartilhado com terceiros, anunciantes, parceiros ou qualquer outra parte. A extensão não contém anúncios e não utiliza ferramentas de rastreamento ou análise de comportamento.

6. Dados sensíveis

Os CPFs e CNS gerados pela extensão são sinteticamente válidos, porém completamente fictícios. Eles não correspondem a pessoas reais e não são obtidos de nenhuma base de dados. São gerados por algoritmo matemático exclusivamente para fins de teste de software.

7. Menores de idade

Esta extensão é voltada para uso profissional em desenvolvimento e QA de software e não coleta dados de nenhum usuário, independentemente da idade.

8. Alterações nesta política

Eventuais atualizações desta política serão publicadas nesta mesma página. Recomendamos que o usuário a consulte periodicamente.

O uso contínuo da extensão após alterações implica aceitação da versão atualizada.
